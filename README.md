## 📝K-POP 익명 커뮤니티 + EC2 서버 연결해보기 
CRUD를 사용하여 익명 커뮤니티 게시판 만들기.
자신의 정보를 공개할 필요 없이 자유롭게 의견을 교환할 수 있는 웹 페이지 만들기.
단, 작성한 사람은 자신의 글을 수정 / 삭제가 가능해야 한다.

## 스택
* Spting Boot 3.2.1
* Spring Data JPA
* SQL lite
* Thymeleaf
* AWS EC2

## 구현
* 게시글 생성, 수정, 삭제
* 댓글 생성, 수정, 삭제
